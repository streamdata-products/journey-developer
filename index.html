This is the developer portal for the Streamdata.io Journey product. It is a an API for developing and managing the API lifecycle. Breaking the lifecycle down into area, stops, and entries accompanied with links, services, and stops for each entry. Providing a structured approach to learning about, and then ultimately crafting your own API lifecycle for use in trainings, workshops, and demonstrations. There is also an OpenAPI, and a Postman Collection available for this API.
